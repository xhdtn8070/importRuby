
# importRuby

> A simple import *.rb File Gem for Sinatra(Ruby)
(Also, 'importRuby' Gem Can be used in normally  Ruby Project)
 

     gem 'importRuby'
     bundle install

and then, write 


     #./config.ru
     require 'ImportRuby'
     ImportRuby.require_all('controller')
